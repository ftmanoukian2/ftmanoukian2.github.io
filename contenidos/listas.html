Listas

La lista es un tipo de dato utilizado para almacenar una colección ordenada de elementos. Estos elementos pueden ser de cualquier tipo de dato, incluyendo números, cadenas de texto, otras listas e incluso elementos más complejos.

Características principales

  • Colección Ordenada: Los elementos en una lista están organizados en un orden específico, lo que significa que cada elemento tiene una posición única dentro de la lista. Esta posición se conoce como índice y comienza desde 0 para el primer elemento de la lista.
  • Flexibilidad de Contenido: Una lista puede contener elementos de diferentes tipos de datos, lo que permite almacenar una variedad de valores en una sola estructura de datos.
  • Mutabilidad: Las listas en Python son mutables, lo que significa que pueden ser modificadas después de su creación. Los elementos de una lista pueden ser agregados, eliminados o modificados según sea necesario.
  • Acceso y Modificación: Los elementos individuales de una lista pueden ser accedidos y modificados mediante su índice. Esto permite realizar operaciones como la lectura de valores, la asignación de nuevos valores y la manipulación de elementos existentes.
  • Iterabilidad: Las listas son iterables, lo que significa que se pueden recorrer fácilmente utilizando bucles for y otras estructuras de control de flujo.

Ejemplos de declaración de listas 

        
# lista con 5 elementos tipo str
frutas = ["manzana", "banana", "naranja", "uva", "pera"]

# lista vacia
lista_vacia = []

# lista de listas
lista_de_listas = [[1,2,3],[4,5,6,7,8]]

Ejemplos de acceso a listas 

        
# acceso en lista simple
frutas[0]               # devuelve 'manzana'
frutas[-1]              # recorre de atrás hacia adelante, devuelve 'pera'

# acceso en lista de listas
lista_de_listas[1]      # devuelve [4,5,6,7,8]
lista_de_listas[1][3]   # devuelve 7

Las listas permiten acceder a un subconjunto de los datos que contienen, dado por un par de índices mínimo y máximo. Opcionalmente, se puede agregar un parámetro correspondiente a la distancia entre índices.

Sintaxis de subconjuntos 

        
lista[minimo:maximo]
lista[minimo:maximo:distancia]
  • minimo: el índice a partir del cual comienza el subconjunto.
  • maximo: el tope superior (no incluido) de índices hasta el cual llega el subconjunto.
  • distancia: la distancia entre los índices con los que se forma el subconjunto. Este parámetro es opcional

En caso de no proveer alguno de los valores, los mismos se autocompletan de la siguiente manera: mínimo 0, máximo -1 y distancia 1.
Esto es especialmente útil si se busca respetar alguno de los extremos de la lista en el subconjunto.

Ejemplos de subconjuntos 

        
lista[::2]      # devuelve una lista con todos los elementos con índice par
lista[:5]       # devuelve los elementos con índice entre 0 y 4 incluídos
lista[10:]      # devuelve los elementos con índice 10 en adelante
lista[::-1]     # devuelve la lista en orden inverso

Métodos de listas

Los métodos son funciones que corresponden a un tipo de dato en particular. Hay múltiples métodos que permiten simplificar la operación con listas, algunos de los cuales se ejemplifican a continuación: 

        
lista.count(elemento)
# devuelve la cantidad de apariciones de 'elemento' en la lista

lista.index(elemento)
# si 'elemento' se encuentra en la lista, devuelve la posicion de la primera aparición

lista.append(elemento)
# agrega 'elemento' al final de la lista

lista.pop(indice)
# remueve el elemento en la posición 'indice', y lo devuelve en forma de retorno

lista.insert(indice,elemento)
# desplaza todos los elementos a partir de 'indice'
# una posición a la derecha, e inserta 'elemento' en su lugar

La mayoría de estos métodos son también aplicables a datos de tipo 'str', que actúan como listas cuyos elementos son los caracteres que las conforman.
