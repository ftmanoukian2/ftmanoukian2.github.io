Condicionales

Los condicionales son estructuras de control que permiten tomar decisiones en base a la evaluación de una condición. Es decir, pueden ejecutar una secuencia de código en caso de que se cumpla una condición, y ejecutar una secuencia diferente o realizar un salto en el caso contrario.

Estructuras y sintaxis

  • Condición inicial

    Un condicional siempre comienza con la condición inicial, que se escribe de la siguiente manera: 

                    
if condicion:
    accion1 # accion dentro del condicional
    ...
accion2 # accion fuera del condicional

    La condición evaluada, en principio, va a ser una expresión dada por un operador de comparación. Los operadores disponibles son: 

                    
== #(igual)
!= #(distinto)
> #(mayor)
< #(menor)
>= #(mayor o igual)
<= #(menor o igual)

    A su vez, las expresiones se pueden combinar entre sí utilizando operadores lógicos o booleanos (and, or, etc.). Es importante, en caso de combinar expresiones, indicar el orden en que se deben realizar las mismas utilizando paréntesis (las expresiones entre paréntesis se evalúan primero).

  • Condición secundaria:

    En caso de que la evaluación inicial devuelva un resultado negativo, se puede evaluar una segunda condición de la siguiente manera: 

                    
if condicion:
    accion1 # accion dentro del condicional
    ...
elif condicion2:
    accion2 # accion dentro de la condición secundaria
    ...
accion3 # accion fuera del condicional

  • Acción alternativa:

    En caso de querer realizar una acción cuando no se cumpla ninguna de las condiciones dadas, se debe realizar de la siguiente manera: 

                    
if condicion:
    accion1 #accion dentro del condicional
    ...
elif condicion2:
    accion2 #accion dentro de la condición secundaria
    ...
else:
    accion3 #accion alternativa
accion4 #accion fuera del condicional
