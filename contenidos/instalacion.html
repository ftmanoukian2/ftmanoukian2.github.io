Instalación del entorno de trabajo

El entorno de trabajo que utilizaremos en la materia consiste en el intérprete de Python y el IDE Visual Studio Code.

Es importante respetar el orden de instalación para lograr configurar el entorno correctamente.

Pasos para la instalación de Python

  1. Ingresar a https://www.python.org/downloads/ y descargar la última versión (3.12.2 hasta la fecha) para Windows con el botón amarillo.
  2. Una vez descargado el instalador, ejecutarlo y seguir los pasos.

Pasos para la instalación de Visual Studio Code

  1. Ingresar a https://code.visualstudio.com/download y descargar la última versión para Windows.
  2. Una vez descargado el instalador, ejecutarlo y seguir los pasos.
  3. Una vez instalado y abierto el programa, se visualizarán opciones de personalización. Recomendamos no cambiar las opciones predeterminadas.
  4. Cerrar las pestañas iniciales, y dirigirse a la sección de extensiones en el menú de la izquierda (en la ilustración de cuatro cuadrados). Dentro de la misma, escribir en el buscador superior "Python" y descargar únicamente la primer opción que aparece (desarrollada por Microsoft). Una vez instalada, cerrar la pestaña.
  5. Presionar las teclas Ctrl+Shift+P, tras lo cual se abrirá un cuadro de texto en la parte superior de la ventana. En el mismo, escribir "Python: select interpreter", presionar Enter en la opción y seleccionar la opción "Python 3.12.2".
  6. Nuevamente presionar Ctrl+Shift+P, ahora escribiendo "Terminal: select default profile" y dentro de esta opción seleccionando "Command Line".

Si alguno de los pasos anteriores no funcionó, pueden contactarnos por el campus.
