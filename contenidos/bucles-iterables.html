Bucles

Los bucles son estructuras de control que permiten ejecutar un bloque de código de forma repetida.
Existen dos tipos de bucle en Python, uno con una cantidad indeterminada de repeticiones - while - y otro con una cantidad determinada - for - .
Denominaremos iteración a cada una de las repeticiones que se da en la ejecución de un bucle.

while

La ejecución de este tipo de bucles está determinada por una condición booleana. La comprobación de esta condición se realiza antes de cada iteración, por lo que el cumplimiento de la condición debe poder darse dentro de una de las iteraciones del ciclo (de lo contrario, el programa nunca podría continuar mas allá del bucle).

Sintaxis 

        
while condicion:
    accion1       # accion realizada de forma repetida

accion2           # accion realizada al finalizar el bucle
Ejemplos 
            
variable = 0

# el siguiente bucle se repite 10 veces
while variable < 10:
    # si la variable no se modificara durante el bucle,
    # el programa no podría continuar más allá.
    variable = variable + 1

    print(variable)

print("bucle finalizado")
            
        
            
# solicito un número al usuario
numero_ingresado = input("Ingresar un número")

# el bucle se inicia sólo si el usuario no ingresó un numero válido.
# se va a repetir hasta que el usuario ingrese un número, 
# solicitando uno nuevo por cada repetición.
while numero_ingresado.isnumeric() == False:
    numero_ingresado = input("No ingresó un número. Por favor ingresar un número")

numero_ingresado = int(numero_ingresado)
print(numero_ingresado)

for

Este tipo de bucles está asociado a un conjunto de elementos agrupados, el cual se denomina iterable. La cantidad de iteraciones a realizar se corresponderá con la cantidad de elementos pertenecientes al iterable, y, en cada iteración, una variable se asociará a cada uno de los elementos de forma individual.

Iterables

La categoría iterable no se asocia a un tipo de dato particular, sino a objetos que están compuestos por elementos individuales, sea cual sea su tipo. Ejemplos de objetos iterables son:

  • Listas (sea cual sea el tipo que contiene, incluyendo listas con elementos de distintos tipos)
  • Strings
  • Tuplas
  • Rangos (range())

Sintaxis 

        
for elemento in objeto_iterable:
    accion1     # accion realizada en todas las iteraciones

accion2         # accion realizada al finalizar el bucle
Ejemplos 
            
    # el siguiente bucle se repite 10 veces, imprimiendo los números 0-9.
    for i in range(10):
        print(i)

La declaración del bucle mostrada en este ejemplo es totalmente válida para los casos en que se requiera repetir un bloque de código una determinada cantidad de veces, por más que no se utilice la variable 'i' (en cuyo caso debe ignorarse dentro del bloque). 

            
    texto = "Hola"
    
    # el siguiente bucle imprime, uno por uno, 
    # los caracteres que conforman la variable "texto"
    for letra in texto:
        print(letra)
