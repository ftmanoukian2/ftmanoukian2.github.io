Instalación de Pyserial

A continuación se presenta la secuencia de pasos a realizar para configurar correctamente la computadora e instalar el módulo "pyserial" en Windows.

Paso 1 - Deshabilitar alias automáticos

  1. Desde el menú Inicio, buscar y abrir la aplicación "Configuración".


  2. Dentro de Configuración, seleccionar "Aplicaciones".


  3. Luego, seleccionar "Aplicaciones y características"


  4. Seleccionar la opción en azul "Alias de ejecución de aplicaciones".


  5. Recorrer la lista hasta encontrar las siguientes opciones:

    • Instalador de aplicación - python.exe
    • Instalador de aplicación - python3.exe

    Deseleccionar ambas opciones, y cerrar la ventana.


Paso 2 (Opcional) - Actualización de Python

  1. Descargar la última versión de Python desde el siguiente enlace.

  2. Abrir el archivo descargado, y seleccionar la opción "Customize Installation"


  3. En la sección "Optional Features", tildar todas las casillas.


  4. En la sección "Advanced Options", tildar las casillas mostradas en la captura.
    No se debe modificar la ruta de instalación automática.
    Al finalizar, presionar el botón "Install" en la esquina inferior derecha.


  5. Una vez finalizada la instalación, se abrirá la siguiente ventana. Presionar el botón "Close".


Paso 3 - Instalar el módulo

  1. Presionar las teclas "Win" + "R" para abrir la ventana "Ejecutar". Escribir "cmd" en el cuadro de texto y presionar "Enter".


  2. En el terminal, escribir el siguiente comando y presionar "Enter":

    python -m pip install pyserial

    Al finalizar la instalación, se imprimirá el siguiente mensaje:


  3. Probar el funcionamiento del módulo:

    1. Con el Arduino desconectado de la computadora, ingresar el siguiente comando en el terminal: 
      python -m serial.tools.list_ports
      Como respuesta, debería recibirse el mensaje: 
      no ports found
    2. Luego, conectar el arduino e ingresar el comando nuevamente. El mensaje de respuesta ahora deberá indicar el puerto en que se conectó el Arduino.
