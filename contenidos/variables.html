Variables

Las variables son contenedores de información que almacenan valores que pueden cambiar durante la ejecución del programa. En Python, las variables se pueden utilizar para almacenar datos como números, cadenas de texto, listas, entre otros más avanzados que veremos más adelante.

Tipos

Los principales tipos de datos vistos hasta el momento son:
  • int

    Almacenan números enteros, tanto positivos como negativos. En Python, no existe un límite para el valor que pueden tomar.

  • float

    Almacenan números de punto flotante; es decir, "con coma". Suelen ser más flexibles para hacer cálculos que las int.

  • str

    Almacenan cadenas de caracteres. Las mismas pueden contener letras, números, signos de puntuación y otros símbolos. Su longitud puede ir desde un único caracter hasta un párrafo completo.

Declaración

Las variables se declaran asignándoles un valor utilizando el operador de asignación (=). Por ejemplo: 

        
x = 10
nombre = "fran"

Los tipos de variable se asignan automáticamente de acuerdo al valor que contienen - es decir, no deben ser declarados por el programa. A su vez, las variables no conservan su tipo de dato durante toda la ejecución del programa sino que el mismo se ajusta al valor que contienen en cada momento. 

        
#solicito información al usuario

entrada_usuario = input() #tipo: str
entrada_usuario = int(entrada_usuario) #tipo: int

Nombres

Los nombres de las variables en Python pueden contener letras, números y guiones bajos, pero deben comenzar con una letra o un guion bajo. No hay límite en la cantidad de variables que se pueden crear en un programa.
